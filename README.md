# LabIntel Platform

A modern, professional healthcare diagnostic platform designed for Patients, Lab Staff, and Administrators.

## 🚀 Overview

LabIntel is a centralized hub for managing medical diagnostic reports and laboratory operations with a focus on visual excellence and user experience.

## ✨ Features

- **Unified Authentication**: A streamlined login portal with role selection (Patient, Lab Staff, Admin).
- **Role-Based Dashboards**:
    - **Patients**: View, download, and manage personal medical reports.
    - **Lab Staff**: Create and upload reports, manage patient records.
    - **Admins**: Monitor system analytics, branch performance, and user activities.
- **Premium UI/UX**: Built with a sophisticated navy-and-teal aesthetic, featuring glassmorphism and smooth transitions.
- **Partner Network**: Integration with top-tier diagnostic labs like Apollo Diagnostics, Dr Lal PathLabs, and Metropolis.

## 🛠️ Tech Stack

- **Frontend**: React.js + Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **Icons**: Lucide React
- **Mock Data**: Robust simulated data layer for development and testing.

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Labintel-dev/Labintel.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/pages`: Main application views (Dashboards, Landing Page, Role Selection).
- `src/data`: Mock data stores for roles and reports.
- `src/App.jsx`: Main routing logic.
- `src/index.css`: Global design system and theme tokens.
